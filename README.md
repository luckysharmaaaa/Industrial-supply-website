# Industrial Supply Website — MERN Stack

A full-stack MERN business website built for an industrial cleaning materials supplier.  
The project includes a responsive frontend, REST API backend, MongoDB Atlas integration, contact form handling, product showcase pages, and modern UI features.

---

# Tech Stack

## Frontend
- React.js
- Vite
- React Router
- Tailwind CSS / CSS

## Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

---

# Features

- Fully responsive design
- Sticky navbar
- Mobile hamburger menu
- Animated hero slider
- Product dropdown menu
- Product showcase pages
- Blog section
- Contact form UI
- REST API integration
- MongoDB Atlas database
- WhatsApp floating button
- Smooth scrolling animations
- Hover effects and modern UI
